# 在线教育管理系统

## 预览项目模板

下载静态页模板到本地：

```bash
$ git clone https://github.com/lipengzhou/admin-template.git
```

安装依赖：

```bash
$ cd admin-template
$ npm install
```

启动预览：

```bash
$ npm start
```
